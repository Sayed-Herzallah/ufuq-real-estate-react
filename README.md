# UFUQ Real Estate

**UFUQ Real Estate** is a modern and fully responsive real estate web platform built with **React**.  
It allows users to browse properties, explore statistics, read client testimonials, and easily get in touch with the team.  

### Key Features
- Clean and intuitive **UI/UX** for an engaging user experience.
- Dynamic **property listings** with detailed information.
- **Multi-language support** using React Context API.
- Interactive sections: Hero, Stats, Testimonials, Why Us, Steps, and Contact.
- Fully responsive design for desktops, tablets, and mobiles.

### Technologies
- **React 18** for a dynamic frontend experience
- **Context API** for state and language management
- **Lucide-react** for icons
- **Custom CSS** for styling and layout
